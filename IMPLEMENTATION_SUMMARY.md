# 덕꾸 Deokkku 구현 정리

## 프로젝트 개요

덕꾸는 애니메이션 감상 기록을 다이어리 꾸미기 형태로 저장하고 공유할 수 있는 웹 기반 아카이브 서비스입니다.

현재 구현은 Django가 백엔드와 페이지 서빙을 담당하고, HTML/CSS/JavaScript/Vue가 프론트엔드 UI와 편집 인터랙션을 담당하는 구조입니다.

## 기술 구조

- Backend: Django
- Frontend: HTML, CSS, JavaScript, Vue 3
- Static assets: Django static 경로 사용
- 실행 주소: `http://127.0.0.1:8000/`

## 주요 파일

- `manage.py`
- `anime_archive/settings.py`
- `anime_archive/urls.py`
- `anime_archive/views.py`
- `anime_archive/templates/index.html`
- `anime_archive/static/css/app.css`
- `anime_archive/static/js/app.js`
- `anime_archive/static/images/main-logo.png`

## 백엔드 구현

Django에서 아래 API를 제공합니다.

- `GET /api/records/`
  - 샘플 감상 기록 목록 반환
- `POST /api/analyze/`
  - 감상문, 제목, 분위기를 기반으로 AI 분석 형태의 키워드/문구 반환
- `POST /api/records/create/`
  - 새 감상 기록 생성 형태의 응답 반환

현재 AI 분석은 실제 외부 API 연동 전 단계의 mock 로직이며, 추후 Gemini API 또는 AniList API 연동을 붙이기 좋은 구조로 분리해두었습니다.

## 로그인 후 레이아웃

구현 내용:

- 왼쪽 사이드바 유지
- 사이드바 메뉴:
  - 홈
  - 내 앨범
  - 보관함
  - 스티커샵
  - 탐색
  - 설정
- 각 사이드바 버튼 클릭 시 세부 페이지 전환
- 홈 화면은 다이어리 편집 화면
- 내 앨범 화면은 저장된 카드 확인 화면

커뮤니티 기능은 MVP 범위에서 제외되어 제거했습니다.

## 다이어리 편집 화면

초기 상태:

- 이름, 날짜, 별점만 표시
- 본문은 빈 다이어리 캔버스
- 샘플 이미지, 샘플 메모, 샘플 태그, 고정 장식 제거
- 캔버스가 비어 있을 때만 안내 카드 표시
- 스티커/이미지/메모 등 요소를 하나라도 추가하면 안내 카드 자동 숨김

다이어리 크기:

- 기존 수첩처럼 좁아 보이던 비율을 개선
- 더 넓은 펼친 앨범형 캔버스로 조정

## 새 기록 생성

`＋ 새 기록` 버튼 클릭 시 작은 팝업 모달이 표시됩니다.

입력 항목:

- 작품명
- 감상 날짜
- 별점

동작:

- `시작하기` 클릭 시 빈 캔버스로 새 기록 시작
- 입력한 작품명, 날짜, 별점이 다이어리 상단에 반영
- 추후 AniList API 자동완성을 붙일 수 있도록 작품명 입력 필드를 독립적으로 구성

모달 개선:

- 스크롤 위치와 관계없이 화면 상단 쪽에 바로 보이도록 수정
- 모달 내부가 길어지면 모달 내부에서만 스크롤

## 스티커샵 및 꾸미기 요소

카테고리:

- 전체
- 스티커
- 프레임
- 말풍선
- 아이콘
- 배경

동작:

- 각 탭 클릭 시 해당 카테고리 요소 표시
- 요소 클릭 시 다이어리 캔버스 위에 추가
- 스티커샵 목록은 고정 높이와 내부 스크롤을 사용해 탭 전환 시 레이아웃 흔들림 방지

기존 스티커 감성은 복구했으며, 귀여운 스티커 요소를 다시 추가했습니다.

## 이미지 첨부

구현 내용:

- 사용자가 원하는 이미지를 직접 첨부 가능
- 첨부 이미지는 다이어리 위에 사진 스티커처럼 추가
- 첨부 이미지는 이동, 삭제, 크기 조절 가능

초기 샘플 이미지는 모두 제거했습니다.

## 텍스트 메모

구현 내용:

- `메모 추가` 버튼으로 텍스트 메모 생성
- 메모 안에 직접 텍스트 입력 가능
- 드래그 이동 가능
- 삭제 가능
- 크기 조절 가능

메모는 종이 메모지처럼 보이도록 스타일링했습니다.

## 요소 선택, 삭제, 크기 조절

대상:

- 스티커
- 프레임
- 말풍선
- 아이콘
- 배경
- 첨부 이미지
- 텍스트 메모

동작:

- 요소를 추가하면 삭제 버튼은 바로 보이지 않음
- 요소를 한 번 클릭하면 해당 요소에만 삭제 버튼 표시
- 선택된 요소에는 크기 조절 슬라이더 표시
- 슬라이더로 요소 크기 조절 가능
- 삭제 버튼 클릭 시 해당 요소만 삭제

## 저장 및 내 앨범

구현 내용:

- 저장 버튼 위치와 가독성 개선
- `실행 취소`, `메모 추가`, `저장` 버튼을 한 줄에 표시
- 저장 버튼 클릭 시 토스트 알림 표시

저장 알림:

- `내 앨범에 저장되었습니다.`

내 앨범 연동:

- 저장 버튼 클릭 시 현재 기록이 `내 앨범` 페이지에 카드로 추가
- 내 앨범 페이지에서 저장된 카드 확인 가능
- 저장 카드에는 날짜, 제목, 별점, 장식 개수 표시

## 모바일 공유 미리보기

구현 내용:

- 웹에서 만든 다이어리 기록을 모바일 공유 형태로 미리보기
- 공유 모드:
  - 스토리 9:16
  - 피드 1:1
  - 링크 카드
  - 이미지 저장

추가한 스티커/이미지 일부가 공유 미리보기에 반영됩니다.
